pymagic9¶

What is this?¶

This is a Python library based on calling the stack of frames at runtime and analyzing the code object of frames. For example, the nameof function is implemented here (as in C#).

Install¶

pip install pymagic9

Support¶

CPython 2.7, 3.6, 3.7, 3.8, 3.9, 3.10 on Windows, Ubuntu and MacOS platform.
